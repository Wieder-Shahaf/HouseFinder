```python
async def send_session_expiry_alert() -> None:
    """Send Web Push alert when Facebook session expires (D-07).

    Hebrew notification:
      Title: "פייסבוק דורש התחברות מחדש"
      Body: "לגתי לדף הכניסה, סריקת קבוצות דולגה"
    """
    try:
        subscription_raw = SUBSCRIPTION_FILE.read_text(encoding="utf-8")
        subscription = json.loads(subscription_raw)
    except (FileNotFoundError, Exception) as exc:
        logger.warning("Session expiry alert: no push subscription — %s", exc)
        return

    payload = {
        "title": "פייסבוק דורש התחברות מחדש",
        "body": "לגתי לדף הכניסה, סריקת קבוצות דולגה",
        "url": settings.app_url,
    }
    try:
        webpush(
            subscription_info=subscription,
            data=json.dumps(payload, ensure_ascii=False),
            vapid_private_key=settings.vapid_private_key,
            vapid_claims={"sub": f"mailto:{settings.vapid_contact_email}"},
        )
        logger.info("Session expiry alert: push sent successfully")
    except WebPushException as exc:
        logger.error("Session expiry alert: webpush() failed — %s", exc)
```

**Module-level constants:**
- `GROUPS_FILE = Path("/data/facebook_groups.json")` (per D-01)
- `SOURCE = "facebook_groups"` (per D-15)
- `FACEBOOK_VERIFY_PROMPT` — copy of `VERIFY_PROMPT` from `verifier.py` but replace "a Hebrew real estate listing from Yad2 (יד2)" with "a Hebrew post from a Facebook apartment rental group. The post may be informal/free-form text." Keep the rest identical.

**Helper functions:**
- `_load_groups() -> list[dict]`: Read `GROUPS_FILE`, parse JSON. If file missing or empty list, log warning, return empty list. Format per D-02: `[{"url": "...", "name": "שכירות חיפה"}]`.
- `async def _load_fb_context(p, session_path: str)`: Create browser + context with saved session. Use `p.chromium.launch(headless=True, **get_proxy_launch_args())`, then `browser.new_context(storage_state=session_path, locale="he-IL", viewport={"width": 1280, "height": 800}, extra_http_headers={"Accept-Language": "he-IL,he;q=0.9"})`. Apply stealth: `await Stealth().apply_stealth_async(context)`. Return `(browser, context)`.
- `async def check_session_health(context) -> bool`: Open new page, go to `https://www.facebook.com/`, wait 2s, check if `page.url` contains "login" or "checkpoint" -> return False. Secondary check: `page.locator('[data-pagelet="LeftRail"]').count() > 0`. Return True if passes both. Wrap in try/except, return False on any error. Always close page in finally.
- `def extract_post_source_id(post_url: str, fallback_text: str = "") -> str`: Regex `r'/(?:permalink|posts)/(\d+)'` on post_url. If match, return group(1). Else hash `(fallback_text.encode())` with sha256 and return first 32 chars.

**Main scraper function:**
`async def run_facebook_groups_scraper(db: AsyncSession) -> ScraperResult`:

1. Initialize `result = ScraperResult(source=SOURCE)`.
2. Load groups via `_load_groups()`. If empty, return result immediately (success=True, 0 counts — per D-03).
3. Check session file existence: `if not Path(settings.facebook_session_path).exists()`, add error "session file not found", return result (success=True — pitfall 1 from research).
4. Start Playwright: `async with async_playwright() as p:`, load context via `_load_fb_context(p, settings.facebook_session_path)`.
5. Check session health (per D-06): `if not await check_session_health(context)`, call `await send_session_expiry_alert()`, append error "session_expired: Facebook session requires re-authentication", close browser, return result (success=True — per D-09).
6. For each group in groups list (with 5-15 second random delay between groups — pitfall 5):
   a. Navigate to group URL with `wait_until="domcontentloaded"`, timeout 30000.
   b. Scroll down 3-5 times (with 2s pauses) to load more posts.
   c. Extract posts: query `page.locator('[role="article"]')`. For each article element:
      - Post text: `element.inner_text()`.
      - Post URL: Look for `a[href*="/permalink/"]` or `a[href*="/posts/"]` inside the article. Extract href. If relative, prepend `https://www.facebook.com`.
      - Poster name: First `h2` or `h3` or strong tag text inside the article (the poster's name link).
      - Post date: Look for `abbr` or `span[id*="jsc"]` with a timestamp attribute, or use text extraction for relative dates.
      - `source_id`: Call `extract_post_source_id(post_url, post_text[:200])`.
   d. Track `result.listings_found += len(posts_in_group)`.
   e. If any error during a group, log and continue to next group (per FBGR-05).
7. After all groups, run LLM verification: Prefix each post text with `"[פוסט מקבוצת פייסבוק: {group_name}]\n"` then call `batch_verify_listings(texts)`. Use standard `merge_llm_fields()` pattern.
8. For verified listings (is_rental=True and confidence >= threshold): INSERT into DB using `sqlite_insert(Listing).on_conflict_do_nothing(index_elements=["source", "source_id"])`. Set `source=SOURCE`, `source_id` from extraction, `raw_data` as JSON of extracted fields, plus all LLM-merged fields. Track inserted/rejected/flagged counts on result.
9. Wrap entire function body in try/except: on any unhandled exception, log it, set `result.success = False`, add error string. Always close browser/context in finally block.
10. Return result.
