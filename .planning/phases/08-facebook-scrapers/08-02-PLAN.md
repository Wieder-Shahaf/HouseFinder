**Module-level constants:**

**Main scraper function:**
`async def run_facebook_marketplace_scraper(db: AsyncSession) -> ScraperResult`:

1. Initialize `result = ScraperResult(source=SOURCE)`.
2. Check session file existence: `if not Path(settings.facebook_session_path).exists()`, add error "session file not found", return result (success=True).
3. Start Playwright: `async with async_playwright() as p:`, load context via `_load_fb_context(p, settings.facebook_session_path)` — import from `facebook_groups.py`.
4. Check session health (per D-06): `if not await check_session_health(context)`, call `await send_session_expiry_alert()`, append error, close browser, return result (success=True per D-09).
5. Navigate to `MARKETPLACE_URL` with `wait_until="domcontentloaded"`, timeout 30000.
6. Scroll down 3-5 times (with 2s pauses) to load more listing cards.
7. Extract listing cards from the page. Marketplace listings are rendered as card-like elements. Strategy:
   a. Primary: Look for `a[href*="/marketplace/item/"]` links — these are the listing cards.
   b. For each listing link element:
      - **Listing URL**: Extract `href` attribute. If relative, prepend `https://www.facebook.com`. Pattern: `/marketplace/item/{numeric_id}/`.
      - **source_id**: Extract numeric ID from URL via `re.search(r'/item/(\d+)', href)`.
      - **Title**: The link text or the first text block inside the card element.
      - **Price**: Look for text matching `₪` or numeric patterns like `X,XXX` within the card. Extract integer price.
      - **Location/Address**: Text following the price, typically the neighborhood or city name.
      - **Post date**: May not be directly visible on cards; if absent, use current datetime.
   c. Fallback: If no `a[href*="/marketplace/item/"]` found, try broader extraction: `page.inner_text("main")` and parse text blocks.
8. Track `result.listings_found = len(extracted_listings)`.
9. Run LLM verification: For each listing, build text as `f"[רישום מפייסבוק מרקטפלייס]\nכותרת: {title}\nמחיר: {price}\nמיקום: {location}\n{description}"`. Call `batch_verify_listings(texts)` then `merge_llm_fields()`.
10. For verified listings: INSERT into DB using `sqlite_insert(Listing).on_conflict_do_nothing(index_elements=["source", "source_id"])`. Set `source=SOURCE`, `source_id` from extraction, `raw_data` as JSON. Track counts.
11. Wrap entire function in try/except: on any unhandled exception, log, set `result.success = False`, add error. Always close browser/context in finally.
12. Return result.
